# CodeMaster 4

Учебное приложение "CodeMaster 4" для обучения разработке сайтов и приложений.

## Что внутри

- HTML, CSS, JavaScript, Java и профессиональная практика.
- Выбор языка: русский, английский, датский, грузинский, немецкий.
- Светлая и тёмная тема.
- Настройки звука и вибрации кнопок.
- Большой набор заданий в каждом уроке.
- Отдельная учебная линия "Создать CodeMaster 4" с рекомендацией сначала пройти полный общий курс.
- Адаптация под компьютер, планшет и смартфон.
- PWA-иконки и инструкции установки на iPhone и Android.

## Запуск

Можно открыть `index.html` напрямую в браузере.

Локальный сервер:

```powershell
node server.js
```

После запуска открыть:

```text
http://127.0.0.1:5173
```
